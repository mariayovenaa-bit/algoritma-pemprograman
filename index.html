Tugas 1 - Dasar Algoritma Pemrograman

1. Contoh Algoritma Dalam Kehidupan Sehari-hari

Mengambil Uang di ATM

  1. Masukkan kartu ATM ke mesin.
  2. Pilih bahasa.
  3. Masukkan PIN.
  4. Pilih menu tarik tunai.
  5. Pilih jumlah uang yang diambil.
  6. Tunggu proses.
  7. Ambil uang yang keluar dari mesin.
  8. Ambil kartu ATM dan struk.

Membuat Kopi Dengan Rasa Manis dan Pahit Yang Pas

  1. Siapkan gelas.
  2. Masukkan 1 sendok kopi.
  3. Masukkan 1–2 sendok gula sesuai selera.
  4. Tuangkan air panas.
  5. Aduk hingga tercampur.
  6. Cicipi kopi.
  7. Jika terlalu pahit, tambahkan gula.
  8. Jika terlalu manis, tambahkan sedikit kopi.
  9. Aduk kembali dan kopi siap diminum.

Menghitung Nilai y = 3x + 6

  1. Masukkan nilai x.
  2. Kalikan x dengan 3.
  3. Tambahkan hasilnya dengan 6.
  4. Hasilnya adalah nilai y.
  5. Tampilkan nilai y.

Contoh:

  • x = 2
  • y = 3(2) + 6
  • y = 6 + 6
  • y = 12

2. Pengertian Algoritma dan Program

Algoritma adalah langkah-langkah yang tersusun secara logis dan sistematis untuk menyelesaikan suatu masalah.

Program adalah kumpulan instruksi yang ditulis dengan bahasa pemrograman yang digunakan komputer untuk menjalankan algoritma.

  • Algoritma = langkah-langkah solusi.
  • Program = algoritma yang ditulis dalam bahasa komputer.

3. Tiga Struktur Dasar Algoritma

Sekuensial (Urutan)

Langkah-langkah dijalankan secara berurutan dari awal sampai akhir.

  • Nyalakan kompor.
  • Rebus air.
  • Masukkan mie.
  • Semua dilakukan secara berurutan.

Pemilihan (Selection)

Algoritma melakukan pilihan berdasarkan kondisi tertentu.

  • Jika hujan → bawa payung.
  • Jika tidak hujan → tidak perlu bawa payung.
  • Biasanya menggunakan IF – ELSE.

Perulangan (Looping)

Langkah tertentu diulang beberapa kali sampai kondisi terpenuhi.

  • Menghitung 1 sampai 10.
  • Mengaduk kopi sampai gula larut.
  • Biasanya menggunakan FOR atau WHILE.

4. Fungsi Bahasa Pemrograman

  • Memberi instruksi kepada komputer.
  • Membuat program atau aplikasi.
  • Mengolah data.
  • Mengotomatisasi pekerjaan.
  • Mengontrol perangkat keras (hardware).

Tugas 2 - Tipe Data dan Operator Java

1. Perbedaan Tipe Data Primitif dan Non-Primitif

  • Tipe Data Primitif adalah tipe data dasar yang langsung menyimpan nilai.
  • Contoh: int, float, char, boolean
  • Tipe Data Non-Primitif adalah tipe data yang menyimpan referensi atau alamat ke objek.
  • Contoh: String, Array, Object

2. Kesalahan Pada Penulisan Variabel

Kode:

Int 5angka;
String nama siswa;
Int static;

Kesalahan:

  • Int → harus ditulis int (huruf kecil).
  • 5angka → nama variabel tidak boleh diawali angka.
  • nama siswa → tidak boleh menggunakan spasi.
  • static → merupakan keyword Java sehingga tidak boleh digunakan sebagai nama variabel.

3. Contoh Deklarasi Tipe Data Array

public class Main {

    public static void main(String[] args) {

        int a = 10;
        int b = 2;

        int hasil = a / b;

        System.out.println(
            "Hasil pembagian: " + hasil
        );
    }

}

4. Operator Aritmatika Dalam Java

Operator Keterangan
+ Penjumlahan (menambahkan dua nilai)
- Pengurangan (mengurangi nilai)
* Perkalian (mengalikan nilai)
/ Pembagian (membagi nilai)
% Modulus (menghasilkan sisa hasil bagi)

5. Implementasi Operator Pembagian Dalam Java

public class Pembagian {

        int a = 20;

        int hasil = a / b;

        System.out.println(
            "Hasil pembagian: " + hasil
        );

    }

}

Tugas 3 - Algoritma Sekuensial Penghitungan Tiket Bioskop

Studi Kasus

Program ini dirancang untuk menghitung total biaya pembelian tiket bioskop berdasarkan jumlah tiket yang dibeli dan harga tiket per unit.

Program ini termasuk contoh Algoritma Sekuensial karena setiap instruksi dijalankan secara berurutan tanpa percabangan maupun perulangan.

Flowchart (Diagram Alir)

  • Mulai
  • Input jumlah tiket
  • Input harga tiket per unit
  • Hitung total harga
  • Tampilkan total harga
  • Selesai

Pseudocode / Deskripsi Program

Algoritma: Penghitungan_Tiket_Bioskop

DEKLARASI:
jumlah_tiket : integer
harga_satuan : integer
total_bayar : integer

DESKRIPSI:
1. Input jumlah_tiket
2. Input harga_satuan
3. Hitung total_bayar =
   jumlah_tiket * harga_satuan
4. Tampilkan total_bayar

Program Python

print("=== SISTEM PEMBELIAN TIKET BIOSKOP ===")

# Input
jumlah_tiket = int(
    input("Masukkan jumlah tiket: ")
)

harga_satuan = int(
    input("Masukkan harga per tiket: ")
)

# Proses
total_bayar = jumlah_tiket * harga_satuan

# Output
print("-" * 35)
print(f"Jumlah Tiket : {jumlah_tiket}")
print(f"Harga Satuan : Rp {harga_satuan}")
print(f"Total Bayar  : Rp {total_bayar}")
print("-" * 35)

print("Terima kasih!")

Penjelasan Program

  1. Pengguna memasukkan jumlah tiket yang dibeli.
  2. Pengguna memasukkan harga tiket per unit.
  3. Program menghitung total biaya dengan rumus:
total_bayar =
jumlah_tiket × harga_satuan
  1. Program menampilkan total biaya yang harus dibayar.

Tugas 5 - Array

1. Apa yang Dimaksud dengan Array?

Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data atau elemen dengan tipe data yang sama ke dalam satu variabel.

Setiap elemen dalam array memiliki posisi yang diidentifikasi oleh indeks yang biasanya dimulai dari angka 0.

2. Jelaskan yang Dimaksud dengan Array 1 Dimensi

Array 1 dimensi adalah jenis array yang paling sederhana, dimana elemen-elemennya disusun secara linear atau satu baris saja.

Untuk mengakses elemen di dalamnya cukup menggunakan satu buah indeks.

Analogi

Seperti deretan loker yang diberi nomor urut 0, 1, 2, 3, dan seterusnya.

3. Bagaimana Cara Mendeklarasikan Array?

Cara mendeklarasikan array bergantung pada bahasa pemrograman yang digunakan.

Contoh di Python

# Array/List kosong
nilai = []

# Array/List berisi data
angka = [10, 20, 30, 40, 50]

Contoh di C++

int nilai[5] = {10, 20, 30, 40, 50};

4. Tipe Data yang Digunakan untuk Mendeklarasikan Array

  • int → Bilangan bulat.
  • float / double → Bilangan pecahan.
  • char → Karakter tunggal.
  • string → Teks atau kumpulan karakter.
  • boolean → Nilai logika (true/false).

Pada bahasa seperti C++ dan Java, seluruh elemen dalam satu array harus memiliki tipe data yang sama.

5. Macam-Macam Array

  1. Array 1 Dimensi
    Array linear yang hanya memiliki satu indeks.
  2. Array 2 Dimensi
    Array yang terdiri dari baris dan kolom, sering disebut matriks.
  3. Array Multi Dimensi
    Array yang memiliki lebih dari dua dimensi, misalnya array 3 dimensi.
